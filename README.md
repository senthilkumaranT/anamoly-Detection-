# Anomaly Detection Simple

A simple tool to find unusual data points (anomalies) in a dataset using machine learning.

## How it works

1. **Detection**: Uses the **Isolation Forest** algorithm to identify outliers.
2. **Labeling**: It marks data points as either `0` (Normal) or `1` (Anomaly).
3. **Visualization**: Shows a plot where anomalies are highlighted in red.

## Files

- `note_book.ipynb`: The code to run the analysis.
- `scatter_plot_dataset.csv`: The data file.

## Requirements

Requires these Python libraries:
- `pandas`
- `matplotlib`
- `scikit-learn`
